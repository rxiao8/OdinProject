Typography & Text-alignment

There are different ways to style the font based on family, size, weight, and alignment. For font family, the first listed will be used unless the browser cannot find or support it. Therefore, it is best practice to have a list of "font familys" and generic family.

Image Edits

By default, the size of the images will be their orignal sizes. One way to resize is to define one dimension and have the others set to auto. A best practice is to explicitly state the height and width even if you don't plan to adjust them since it will leave a blank space if the image was not able to load properly. The blank space is reserved.
